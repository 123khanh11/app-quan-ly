## 🔄 CẬP NHẬT TRONG TƯ TƯƠNG LAI

Nếu bạn sửa code:

```bash

flutter build web --release

# 2. Deploy lên Vercel
vercel --prod

# Vercel sẽ tự động cập nhật!
```

## 💡 TIPS

### 1. Chia Sẻ URL
```
Gửi link này cho mọi người:
https://[URL-của-bạn].vercel.app
```

### 2. Custom Domain
```
Nếu bạn muốn domain riêng:
1. Mua domain (GoDaddy, Namecheap, v.v.)
2. Vercel Dashboard → Domains
3. Thêm domain của bạn
```

### 3. Xem Logs
```
vercel logs --prod
```

### 4. Redeploy
```
vercel redeploy
```

## 🆘 NẾU GẶP VẤN ĐỀ

### "Lỗi 404"
```
Có thể vercel.json chưa được setup đúng
→ Kiểm tra vercel.json trong thư mục gốc
```

### "Lỗi Login"
```
→ Kiểm tra Supabase credentials
→ Mở DevTools Console (F12)
→ Xem message lỗi
```

### "Trang trắng"
```
→ F5 refresh
→ Xóa cache (Ctrl+Shift+Delete)
→ Thử browser khác
```
