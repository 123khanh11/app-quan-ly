## ⚡ BƯỚC NHANH NHẤT (2 PHÚT)

### 1️⃣ Tạo Tài Khoản

**Cách A: SQL Script (Khuyến nghị)**
```
1. Mở: https://supabase.com/dashboard
2. Đăng nhập tài khoản Supabase
3. SQL Editor → + New Query
4. Mở file: SQL_TAO_TAI_KHOAN.sql
5. Copy toàn bộ nội dung
6. Paste vào SQL Editor
7. Click Run (Ctrl+Enter)
```

**Cách B: Dashboard UI**
```
1. https://supabase.com/dashboard
2. Authentication → Users
3. Add user
4. Email: admin@example.com
5. Password: password123
6. Save
```

### 2️⃣ Đăng Nhập Ứng Dụng

```
1. Mở: http://localhost:8000
2. Email: admin@example.com
3. Password: password123
4. Click Đăng Nhập
```

### 3️⃣ ✓ Hoàn Thành!

Bạn đã vào ứng dụng! 🎉
