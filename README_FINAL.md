## 🚀 3 BƯỚC ĐỂ BẮTĐẦU

### 1️⃣ **TẠO TÀI KHOẢN**

**Mở Supabase Dashboard:**
```
https://supabase.com/dashboard
```

**Vào Authentication:**
```
Menu bên trái → Authentication → Users
```

**Thêm User:**
```
Click "Add user"
Email: admin@example.com
Password: password123
Click "Save"
```

### 2️⃣ **MỞ ỨNG DỤNG**

```
http://localhost:8000
```

### 3️⃣ **ĐĂNG NHẬP**

```
Email: admin@example.com
Password: password123
Click "Đăng Nhập"
```
