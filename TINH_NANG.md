# Tính năng chi tiết

## 📊 Dashboard
- Tổng quan doanh thu và đơn hàng
- Thống kê theo trạng thái đơn hàng
- Biểu đồ trực quan
- Cập nhật realtime

- ✅ Thêm/Sửa/Xóa sản phẩm

- ✅ Phân loại theo danh mục, thương hiệu

- ✅ Trạng thái kích hoạt/vô hiệu hóa
- ⏳ Upload hình ảnh sản phẩm (sắp có)
- ⏳ Quản lý biến thể (size, màu sắc)
